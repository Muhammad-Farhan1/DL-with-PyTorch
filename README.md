# 🧠 Deep Learning with PyTorch

This repository contains my hands-on implementation of **Deep Learning concepts using PyTorch**. It includes practical notebooks, projects, and experiments covering the complete workflow of building, training, and evaluating deep learning models.

---

## 🚀 About the Repository

This repository is designed to strengthen my understanding of **Deep Learning** by implementing models from scratch using **PyTorch** — a popular open-source deep learning framework known for its flexibility and dynamic computation graph. ([Wikipedia][1])

The goal is to move from **fundamentals → real-world projects**, focusing on both theory and practical implementation.

## 🛠️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Muhammad-Farhan1/DL-with-PyTorch.git
```

2. Navigate to the project folder:

```bash
cd DL-with-PyTorch
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run Jupyter Notebook:

```bash
jupyter notebook
```
